# Jorero Store Tech Stack

This project is intentionally kept simple and practical:

- Node.js
- Express.js
- EJS templates
- MongoDB with Mongoose
- HTML
- CSS
- Vanilla JavaScript

No React, Next.js, Vite, Tailwind build step, or frontend framework is required.

## Run locally

```bash
npm install
cp .env.example .env
npm run seed
npm start
```

Then open:

```text
http://localhost:3000
```

Admin:

```text
http://localhost:3000/admin/login
```
